# Stock Analysis Dashboard

A Python-based stock analysis and visualization project that collects stock market data, processes it, and presents insights through an interactive dashboard.

## Features

* Fetch and analyze stock market data
* Store stock information in JSON format
* Interactive dashboard for visualization
* Track historical stock performance
* Clean and simple dashboard interface
* Expandable structure for future enhancements

## Project Structure

```text
Stock_analysis/
│
├── StockAnalysis.py
├── stock_data.json
├── stockdashboard.html
├── README.md
```

## Installation

Clone the repository:

```bash
git clone https://github.com/pratapmishra123/Stock_analysis.git
```

Move into the project directory:

```bash
cd Stock_analysis
```

Install required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the analysis script:

```bash
python StockAnalysis.py
```

Open the dashboard file in your browser:

```bash
stockdashboard.html
```

## Workflow

1. Run the Python script
2. Fetch stock data
3. Store data in JSON format
4. Generate dashboard output
5. View the results
